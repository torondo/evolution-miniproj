# evolution-miniproj

This is the code for the Evolution module's miniproject. The project is a part of fulfilment of Imperial College London's Living Planet with Ecology, Evolution and Conservation MSc degree.

Go to the folder [gj_code](https://github.com/torondo/evolution-miniproj/tree/master/gj_code) to view the markdown and download all necessary RDS files. 
The markdown `gj_code.md` is meant to be viewable in GitHub, while the folder [downloadme](https://github.com/torondo/evolution-miniproj/tree/master/gj_code/downloadme) contains the file `gj_code.Rmd` and RDS files key for running the code in RStudio.
